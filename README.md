
# Storybook sandboxes of all kinds

  

Storybook: https://storybook.js.org/

  

### Current projects

  

Angular

```
cd angular-stories
yarn install

# to run Angular app
yarn start

# to run Angular Storybook
yarn storybook
```

Vue

```
cd vue-stories
yarn install

# to run Vue app
yarn start

# to run Vue Storybook
yarn storybook
```
